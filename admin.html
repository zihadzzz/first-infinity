🔧 Firebase Setup Guide
1
Firebase Project তৈরি করো
console.firebase.google.com এ যাও → Add project → নাম দাও "First Infinity" → Create
2
Authentication Enable
Left menu → Authentication → Get Started → Email/Password → Enable → Save
3
Firestore Database
Left menu → Firestore Database → Create Database → Start in test mode → Location: asia-south1 → Create
4
Storage Enable
Left menu → Storage → Get Started → Start in test mode → Done
5
Config Copy করো
⚙️ Project Settings → Your Apps → Web App (</>) → Register → firebaseConfig values copy করো
🔑 তোমার Firebase Config
Firebase Console থেকে এই values নিয়ে এখানে দাও। Save হয়ে থাকবে।
📱 App Deploy করো (Netlify)
1. netlify.com/drop এ যাও
2. FirstInfinity.html drag করে drop করো
3. Link পাবা → সেটা বন্ধুদের দাও ✅
4. আলাদাভাবে FirstInfinity_Admin.html ও deploy করো
5. Admin link শুধু তুমি use করবা, কাউকে দিও না!
